Laboratoriya İşləri və Texniki Hesabatlar:

Bu repozitoriya "İnformasiya Təhlükəsizliyi" ixtisası üzrə apardığım laboratoriya işlərini, sistem analizi (Reverse Engineering), şəbəkə təhlili və kriminalistik (Forensics) tədqiqat hesabatlarını əhatə edir.

Layihələr və Hesabatlar
Aşağıdakı mövzular üzrə hazırlanmış texniki hesabatları (PDF) bu repozitoriyada tapa bilərsiniz:

Forensic Analysis: Sistem fayllarının analizi və zərərli kodların tədqiqi.

Access Control: Role-Based Access Control (RBAC) sistemlərinin tətbiqi və təhlükəsizlik konfiqurasiyaları.
